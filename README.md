    # TraitForge - A Combinatorial Image Generator

    TraitForge is a simple Windows (.NET) desktop application written in C# that generates all possible image combinations from a given set of layered traits. It's designed for artists and creators, especially for projects like generating NFT collections or layered character sprites.

    ### The Story & Current Status

    This application was originally developed as a custom tool for a client. The core functionality is complete and working, but the project was halted before any significant UI/UX refinement.

    As such, the Graphical User Interface is very basic and "developer-focused". **Functionality was prioritized over aesthetics.**

    I am publishing it as a portfolio piece to showcase my skills in C# and .NET desktop development.
